# 标书资料库管理工具 v2.6.2

**自动将公司资料库内容插入 Word/WPS 标书文档**

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 📂 资料库管理 | 创建/编辑/删除条目，支持文本/图片多种格式 |
| 📥 老标书提取 | 从 .docx/.pdf/.txt 中按段落/页码/标题等级自动提取 |
| 📷 图片附件 | 支持插入资质证书、合同扫描件等图片 |
| 📋 批量插入 | 勾选多个条目，一键插入到目标 Word 文档末尾 |
| 🔍 分类+搜索 | 按分类组织，支持关键词全文搜索，支持"未分类"快速筛选 |
| 📊 版本对比 | 查看条目的历史版本，差异高亮显示 |
| 🏷️ 关键词标签 | 8色关键词标签 + 优先级着色 |
| 📄 标书模板管理 | 多人共享资料库，支持模板导入导出 |
| 📋 批量管理 | 批量导入多个老标书、导出资料库为 Word 备份 |

---

## 快速启动

双击运行：
```
dist\标书资料库管理工具\标书资料库管理工具.exe
```

或从源码运行（需 Python 3.12）：
```
python main_gui.py
```

## 使用方法

### 1. 从老标书导入资料
1. 点击顶部 **导入文件** 按钮
2. 选择 .docx / .pdf / .txt 文件
3. 选择解析模式（按段落 / 按页码 / 按标题等级）
4. 在预览窗口中勾选要入库的段落，编辑关键词标签
5. 点击 **开始入库**

### 2. 手动添加条目
- 点击侧边栏 **+ 新建条目**
- 输入标题、正文内容，选择分类、类型（正文/表格/图片说明）

### 3. 管理分类
- 底部分类树支持增/删/重命名
- 侧边栏可快速筛选"全部条目"或"未分类"条目

### 4. 添加图片附件
- 选中条目后，在右侧详情面板点击 **加图片**
- 支持调整图片顺序（↑↓按钮）和删除（×按钮）

### 5. 将资料插入标书
1. 打开目标 Word 文档（.docx）
2. 点击顶部 **插入文档** 按钮
3. 勾选要插入的条目
4. 点击"确定插入"

> 支持 **WPS** 和 **Microsoft Office**，自动识别 WPS 中文样式（标题 1、目录 1 等）

### 6. 修改条目分类
- 双击条目打开编辑对话框，在分类下拉框中选择新分类
- 侧边栏实时刷新显示

### 7. 导出日志
- 点击顶部 **导出日志**，生成含调试信息的 .log 文件

## 文件结构

```
tender_lib/
├── main_gui.py         # 主程序 GUI（含所有窗口类）
├── tender_lib_db2.py   # 数据库操作模块
├── extractors.py       # 文档解析模块（WPS/Word/PDF/TXT）
├── docx_inserter.py    # Word 插入模块
├── tender_lib_db.py    # 数据库基础模块（旧））
├── README.md           # 本文档
├── tender_lib.spec     # PyInstaller 配置
└── dist/
    └── 标书资料库管理工具/
        ├── 标书资料库管理工具.exe  # 最终可执行文件
        └── _internal/              # 依赖文件
```

数据库存储在：`%APPDATA%\tender_lib\tender_library.db`

---

## 数据安全

- 所有数据存储在本地 SQLite 数据库，不上传云端
- 修改/删除条目有历史版本记录
- 导出日志方便排查问题
